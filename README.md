# Job-Portal-Management-System
Full Stack Job Portal Management System built using Spring Boot, Spring Security, Thymeleaf, and MySQL.
## Features

### Job Seeker
- Registration & Login
- OTP Verification
- Resume Upload
- Job Search & Filtering
- Apply for Jobs
- Application Tracking
- Email Notifications

### Employer/Admin
- Post Jobs
- Manage Applications
- Shortlist/Reject Candidates
- Dashboard Analytics
- User Management

## Tech Stack

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- HTML
- CSS
- JavaScript
